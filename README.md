# 🐍 Python - Trabalhos SENAC

Este repositório reúne exercícios e atividades desenvolvidas durante meus estudos de **Python** no **SENAC**.

O objetivo é praticar os principais conceitos da linguagem, como entrada e saída de dados, operadores, estruturas de decisão, listas e lógica de programação.

## 🚀 Como executar

1. Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/python-trabalhos-senac.git
```

2. Entre na pasta:

```bash
cd python-trabalhos-senac/python
```

3. Execute qualquer arquivo:

```bash
python nome_do_arquivo.py
```

Exemplo:

```bash
python cadastro.py
```
